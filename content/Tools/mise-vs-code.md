1. Install **Mise VSCode Extension**[^HuguesVerlin-Webpage-4DBH5GTX] from Extensions
2. If prompted
	>"Mise extension works best with a TOML extension for syntax highlighting and autocompletion. Either of the following are recommended:" Tombi TOML; Even Better TOML

	\> Then install **Even Better TOML Extension**[^FerencTamas-Webpage-QBJKM4YY] from Extensions
3. **Open Command Palette (Ctrl+Shift+P)**\
	\> "Mise: Focus on Tools View"\
	\> and follow this guide[^HuguesVerlin-Webpage-4DBH5GTX]
